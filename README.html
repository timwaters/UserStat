UserStat

A collection of tools to visualize OpenStreetMap contributor behaviour.

UserStat analyzes OpenStreetMap data and summarises it in various ways. OpenStreetMap data is ©OpenStreetMap and Contributors and is licensed under the OpenStreetMap License.
